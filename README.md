Deskripsi Project

Project ini merupakan implementasi dari Ujian Tengah Semester mata kuliah Pemrograman Web 1, yang bertujuan untuk mengonversi desain Figma menjadi website fungsional menggunakan HTML, CSS, dan JavaScript DOM.
Website terdiri dari tiga halaman utama:

1. Halaman Login (index.html)

Halaman ini berisi form login dengan input email dan password (NIM).
Fitur yang diimplementasikan:

Validasi input tidak boleh kosong.

Menampilkan pesan error jika form kosong.

Jika valid, menampilkan pesan “Login berhasil”.

Redirect otomatis ke halaman dashboard.

2. Halaman Dashboard (dashboard.html)

Halaman ini menampilkan ringkasan data menggunakan card summary.
Fitur:

Menampilkan total produk, total penjualan, dan total revenue.

Data diambil dari array object di script.js.

Menggunakan Flexbox untuk tata letak card.

Tombol “Lihat Data Produk” yang mengarahkan ke halaman produk.

3. Halaman List Data Produk (products.html)

Halaman ini menampilkan daftar produk dalam bentuk tabel.
Fitur:

Data produk diambil dari array of object.

Tabel ditampilkan menggunakan perulangan JavaScript (forEach/map).

Aksi:

Edit → menampilkan alert nama produk.

Delete → menghapus baris produk langsung dari DOM dengan konfirmasi.
